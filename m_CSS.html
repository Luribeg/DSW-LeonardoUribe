¿Qué es CSS?

Historia de CSS
Primero nació el HTML, pero con el tiempo surgió la necesidad de darle apariencia a los documentos HTML y con CSS se logró este objetivo. En este video veremos una breve historia de cómo surgió este lenguaje.

Debemos tener claro que el HTML es sólo para estructurar un documento WEB y con las hojas de estilo se cambia la apariencia del documento, en este video vamos a aprender la sintaxis como se emplea este lenguaje, la forma como se crean las reglas y como la herencia afecta cuando se aplica un estilo.
